## Quick Setup

Add `many_lints` to your `analysis_options.yaml`:

```yaml
plugins:
  many_lints: ^0.4.0
```

That's it — the analysis server will automatically download and resolve the plugin from [pub.dev](https://pub.dev/packages/many_lints).

:::note
Requires Dart 3.10+ (Flutter 3.38+)
:::
